# Déploiement local

## Frontend

```
npm run build
npm run preview
```

## Backend (optionnel)

```
npm run server:start
```

## Sécurité

- Les en-têtes de sécurité sont gérés côté hébergeur; en local, testez les fonctionnalités, pas les headers.
- CSP: vérifiez le bon fonctionnement sans inline styles/scripts.
