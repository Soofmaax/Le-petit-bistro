# Installation

## Prérequis

- Node.js 18+ (LTS)
- npm 9+

## Étapes

1. Installer les dépendances:
   ```
   npm install
   ```
2. (Optionnel) Télécharger et optimiser les images:
   ```
   npm run setup:images
   npm run optimize:images
   ```
3. Lancer le serveur de développement:
   ```
   npm run dev
   ```
